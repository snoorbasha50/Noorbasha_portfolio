Who I Am

I'm a Software Development Engineer II based in Bangalore, building next-generation fintech infrastructure at Neokred Technologies.

I architected the full migration from a monolithic MERN stack to a distributed TypeScript microservices platform — integrating Kafka for real-time event streaming, gRPC for inter-service communication, and Redis for high-performance caching across services.

My production work includes engineering 10+ bank integrations with idempotency and fault tolerance, zero-downtime migration of 10M+ records using Apache NiFi with rollback strategies, and a callback retry mechanism with exponential backoff for payment systems.

Outside of the distributed systems world, I build AI-powered applications — including a deployed RAG-based finance dashboard using LLaMA 3.3 70B, BM25 retrieval, and Kafka event streaming. I lead code reviews, drive architectural decisions, and mentor junior engineers.
