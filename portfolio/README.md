# Noorbasha Shaik — Portfolio

Dark minimal portfolio for SDE-2 Full Stack Engineer.

## 🚀 Deploy to Vercel in 2 minutes

### Option 1 — Vercel CLI (fastest)
```bash
npm i -g vercel
vercel
# Follow prompts → done. You get a live URL instantly.
```

### Option 2 — Vercel Dashboard
1. Push this folder to a GitHub repo (e.g. `portfolio`)
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import the repo → Deploy
4. Live at `https://your-project.vercel.app`
